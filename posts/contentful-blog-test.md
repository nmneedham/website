This post was migrated out of the older Contentful-based setup into the new Markdown workflow.

This was originally a quick test post for the newer blog service I was using at the time. The main idea was that API calls and JavaScript were much easier than opening an IDE and hand-writing a new HTML page every single time I wanted to post something.

At the time, that felt like a huge quality-of-life improvement.

Ironically, this post now lives in a Markdown-based system, which is even simpler to maintain going forward.
